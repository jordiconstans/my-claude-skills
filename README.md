# My Claude Skills

Custom skills for Claude AI agents - Providing specialized capabilities and behavioral customizations.

## Overview

This repository contains custom skills designed to enhance Claude's capabilities when integrated with supported AI agents and development tools.

## Skills Included

### Professional Communicator
- **Description**: Makes Claude communicate in a professional, polished manner
- - **Usage**: Install with `npx skills add jordiconstans/my-claude-skills --skill professional-communicator`
  - - **Features**:
    -   - Formal business language
        -   - Structured responses
            -   - Clear and concise explanations
             
                - ## Installation
             
                - To install any skill from this repository:
             
                - ```bash
                  npx skills add jordiconstans/my-claude-skills --skill <skill-name>
                  ```

                  Replace `<skill-name>` with the specific skill you want to install.

                  ## Compatibility

                  These skills work with:
                  - Claude Code
                  - - Cursor
                    - - Windsurf
                      - - Other compatible AI agents
